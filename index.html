Cargar

An uber like app.

We built an uber like app in collaboration with the Cargar Team which was a local startup trying to introduce the bike taxis in Pakistan. Unlike uber, this app did not have different modules for driver and customer. Same app was used by customers and drivers.

Customer Module

App contained a registration process and authentication process for redirecting the user to the right module. In Customer module, user can maintain a profile, add promo codes, order cabs and send an email from within the app to customer support. This app also had a security feature which let the relatives or I.C.E numbers to track the rides in real time. It also had shortcuts to send the alert message to I.C.E numbers in case of emergency. Customers could also rate the ride at the end of it. This was a fully functional module without any cut corners.

Driver Module

Driver module was a fully functional module for drivers. Drivers could attend the ride calls, check the location of rides, check their previous rides and ratings on them, report customers and end rides and much more. All possible features were added into this module.

Backend Server

Backend server would communicate with the firebase database, customer and driver side to handle the job scheduling. It decided based on the distances that which driver should get the call.

Sword

An app which detect firearms by using your phone camera.

We helped Nand Logic, a US based startup build their revolutionary device for security by the name of Sword. This device is used to scan firearms on the go. You can put it up on a drone and fly it over crowd to detect the firearms.

IOS App

This algorithm was mainly developed for IOS. The device uses a Flir Thermal Camera which provides upto 8-10 FPS of thermal images. Our custom built algorithm provides upto 12 FPS and can operate in real time with the Flir Thermal Camera.

Android App

Same Algorithm was ported to android and was optimized 1800% to work in real time on android as well. Now both the Android and IOS users can use this algorithm to detect firearms or dangerous items in their events.

Echo

Speech/Signal based alert system

We created a smart alert system which can be installed in any place. If you are in bank or some difficult place, just say the word "Echo" and the system will sound an alarm and in case of gunshot, system will automatically detect a gunshot and sound an alarm. In the pictures the UI of the app is shown. It was a very easy to use UI with just one button. Once you go out of your home, start the app, click on the start button and have your personal security in your pocket.

Keyword Detection

Keyword is detected by mobile device, hence where ever you are in a building, you just need to have the cellphone with the app in your hand to sound the alarm

Mobile Shortcuts

It can be difficult to start the app during threatening conditions. We have installed mobile shortcuts like pressing volume button three times followed by power button, which activates the app in the background.

Robust Recognition System

Our system can detect gunshots in noisy environments with upto 130 DBs of noise under 500 meters range, accurately

Real Time Mobile Performance

System is built for mobile and can process a clip of 4 seconds in under 2 seconds which means that it can detect gunshots in real time.
